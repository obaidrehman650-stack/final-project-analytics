# Youth Unemployment Global Analysis

## Project Overview
This project analyzes global youth unemployment trends to understand patterns across different countries and years. It applies statistical modeling and time-series forecasting to predict future unemployment rates, providing insights into economic stability and workforce development.

## Objectives
- **Data Exploration:** Analyze the distribution of youth unemployment across regions and time.
- **Trend Analysis:** Identify global and country-specific trends.
- **Forecasting:** Build predictive models to forecast future unemployment rates.
- **Interactive Dashboard:** specific visualization of key metrics.
